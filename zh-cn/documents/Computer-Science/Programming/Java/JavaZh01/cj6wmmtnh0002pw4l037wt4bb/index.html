首先，我们新建了一个名叫test的类（Class）。一般来说，每个类都会是一个单独的文件，并且类名和文件名是严格相同的。但在同一个文件当中包含多个类也并不违反Java的语法。

在test类当中我们创建了两个方法：helloworld()和main(String[] args)。

在进一步讲下去之前，让我们先简要介绍一下方法的构成。

方法前的private/public和static是方法的修饰符。除了修饰方法之外，修饰符也能定义类和变量。

在Java当中，修饰符有两类，即：访问修饰符与非访问修饰符。

访问修饰符包括：public、protected、private和default（即没有）。//TODO 补全表格

非访问修饰符则包括：static、final、abstract以及主要用来修饰多线程代码的synchronized和volatile。

随后的String/void则是返回值的类型。void关键字表示不返回任何值（所以结尾没有return），String关键字则表示返回一个字符串。

每个方法的最后总跟着一对儿圆括号。在该括号里的即是要传递给这个方法的参数。由于并不是每个方法都需要传入参数（本例当中创建的两个方法都没有任何传入参数）。

最后，main(String[] args)方法是主方法，与C++的main()函数类似。所以说，每个Java程序都应该包含这个方法。

接着，我们继续分析。

在helloworld()方法当中，我们声明了一个名为str的字符串（String）型变量，然后我们将str赋值（当然，你也可以选择在声明的时候赋值）为”hello, world”。最后，我们返回str。

在main(String[] args)方法当中，我们调用了Java自带的输出方法：System.out.println()。这里，我们把helloworld()方法（返回值是str）传递给System.out.println()进行输出。
